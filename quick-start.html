üöÄ Quick Start Guide

Create your first attribute system in under 5 minutes!

1 Open the Wizard
Navigate to Window ‚Üí LF Attribute Forge ‚Üí Attribute Data Wizard
2 Configure Data Info
  • Data Name: "Character Attributes"
  • Namespace: "Game.Attributes"
  • Class Prefix: "Character"
3 Add Basic Attributes
Click "Add New" and create these essential attributes:

Health (Vital)

  • Code: HP
  • Value: 100
  • Can Regenerate: Yes
  • Regen Rate: 1.0/sec

Strength (Basic)

  • Code: STR
  • Value: 10
  • Min: 1, Max: 100
4 Generate Code
Click "Next" through Code Settings, then hit "Generate"
üéâ Complete! You now have:
  • CharacterAttributeEnum.cs
  • CharacterAttributes.cs
  • CharacterAttributeData.cs
  • CharacterUIBinding.cs

Add to GameObject

  1. Create an empty GameObject
  2. Add the generated CharacterAttributes component
  3. See your attributes in the inspector!

Quick Runtime Test

// Get component and test basic functionality var attrs = GetComponent<CharacterAttributes>(); // Damage the player attrs.ModifyAttribute("Health", -25f); Debug.Log($"Health: {attrs.health.GetCurrentValue()}"); // Check if alive if (attrs.health.IsAtMin()) { Debug.Log("Player died!"); }
